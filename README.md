# 📊 Regression Models in Machine Learning (Beginner Project)

## 📌 Project Description

This repository contains my beginner Machine Learning projects using different types of regression models.
The purpose of this project is to understand how different regression techniques work and how they can be used to make predictions.

---

## 📂 Models Included

## 📁 Project Files

* **simple_linear_regression.ipynb**
  Contains implementation of Simple Linear Regression.

* **multiple_linear_regression.ipynb**
  Contains implementation of Multiple Linear Regression.

* **polynomial_regression.ipynb**
  Contains implementation of Polynomial Regression.

### 1️⃣ Simple Linear Regression

This model is used to predict a value based on a single independent variable.

Example:

* Predicting student marks based on number of hours studied.

---

### 2️⃣ Multiple Linear Regression

This model is used when there are multiple independent variables affecting the output.

Example:

* Predicting profits based on more than one input variable (R&D Spend, Administration, Marketing Spend,State).

---

### 3️⃣ Polynomial Regression

This model is used when the relationship between variables is not linear.

Example:

* Predicting values when data follows a curved pattern between level of position and salary.
